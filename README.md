# YAPP

Yet Another Python Project in the university.

## Getting Started

### Linux

1. Clone and navigate to this repo.
2. `bash init.sh`
3. Visit `http://localhost:5000/`.

### Window

1. Clone and navigate to this repo.
2. `./init.bat`
3. Visit `http://localhost:5000/`.
